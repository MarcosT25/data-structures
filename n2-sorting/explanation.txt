Ordenar:
int, float, strings, structs

Em C, strcmp("str1", "str2") retorna 0 se forem igual, positivo se str1 > str2 e negativo se str2 > str1

Selection Sort
Passo a Passo
1. Descubra a posição do maior elemento
2. Troque o maior elemento com o último elemento
3. n--
4. Repete o processo com um elemento a menos
